Orbit Project

Lightweight, distributed container orchestration for the edge.

The Mission

Orbit is designed to run modern workloads on "forgotten" hardware. Using highly optimized Docker images and a custom communication layer, it turns old laptops and cheap VPS instances into a powerful, unified cluster.
